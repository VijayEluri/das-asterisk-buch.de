Usage: dahdi set swgain <rx|tx> <chan#> <gain>
        Sets the software gain on a a given channel, overriding the
   value provided at module loadtime, whether the channel is in
   use or not.  Changes take effect immediately.
   <rx|tx> which direction do you want to change (relative to our module)
   <chan num> is the channel number relative to the device
   <gain> is the gain in dB (e.g. -3.5 for -3.5dB)

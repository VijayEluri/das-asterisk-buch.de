wurden mit folgenden Asterisk-Versionen generiert:

1.2: 1.2.30.2

Die Hilfetexte der CLI-Befehle für Asterisk 1.6 sind un-
vollständig.
Leider ist 1.6 diesbezüglich Mist.
Viele Befehle sind in der Liste abgeschnitten:
---cut---
*CLI> help ael
                    ael reload Reload AEL configuration
ael set debug {read|tokens|mac Enable AEL debugging flags
---cut---
Außerdem werden in der Liste auf unterschiedliche Weise
"{...|...|...}" und "[...|...|...]" verwendet.
